Description

Two spots seems like a rather modest size for a parking lot, so let's increase the number of parking spaces. We'll jump straight to having 20 spots, numbered from 1 to 20. Initially, all the spots are vacant.

When the user wants to park the car, the program should find a parking spot with the lowest parking number.

The user can write commands park and leave repeatedly and type exit to end the program.

If the parking lot is filled and there's no room left, the program should type Sorry, parking lot is full..

If there are several available spots for a car, the program should always assign the spot with the lowest number.

Example

 The symbol > represents the user input. Note that it's not part of the input.

> park KA-01-HH-9999 White
White car parked on the spot 1.
> park KA-01-HH-3672 Green
Green car parked on the spot 2.
...

> park Rs-P-N-21 Red
Sorry, parking lot is full.
> leave 1
Spot 1 is free.
> park Rs-P-N-21 Red
Red car parked on the spot 1.
> exit
